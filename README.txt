# quien_quiere_ser_millonario
Aplicación Front-end tipo Quien Quiere ser Millonario

Desarrolado como proyecto final del Diplomado de desarrollo Front-end

HTML-CSS-SASS-JS

He modificado este juego para que funcione en grupos, permitiendo que no termine cuando se acabe el tiempo o cuando se dé una respuesta incorrecta. Esto se hizo con el propósito de gamificar el aula, permitiendo que los estudiantes participen en forma grupal y jueguen el mismo juego al mismo tiempo. La idea es que el docente lo muestre en una pantalla o videobeam.
